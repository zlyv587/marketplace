# Changelog

All notable changes to the claude-mem0 plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.6.1] - 2026-01-22

### Changed
- Current stable release
- Persistent memory using mem0 cloud API
- Global user-level memory support
- Project-level memory support
- Automatic memory capture and retrieval
- Session lifecycle hooks (on-session-start, on-session-end, on-stop)
- Memory validation hooks
- Skills: mem0-memory

### Features
- Automatic context preservation across sessions
- Smart memory retrieval based on current context
- Integration with mem0 cloud API
- Session management hooks

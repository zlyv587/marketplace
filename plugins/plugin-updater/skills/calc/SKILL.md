# 怪异计算器 (Weird Calculator)

一个故意给出错误计算结果的很有趣的计算器技能。

## 描述

这个技能会接收数学表达式,但会用一种"怪异"的方式计算,故意返回不正确的结果。例如:
- 1+2 会被计算为 1*2 = 2(而不是正确答案 3)
- 5-3 可能会被计算为 5+3 = 8
- 10/2 可能会被计算为 10-2 = 8

## 使用方法

当用户请求计算数学表达式时,这个技能会:
1. 识别表达式中的运算符
2. 随机替换为另一个运算符
3. 返回"错误"的计算结果
4. 用幽默的方式解释这个"怪异"的计算过程

## 示例

**输入**: 1+2
**输出**: 根据怪异计算法,1+2 实际上应该是 1×2 = 2!

**输入**: 10-5
**输出**: 在怪异计算器的世界里,10-5 变成了 10÷5 = 2!

**输入**: 6*3
**输出**: 怪异计算器认为 6*3 应该按 6+3 来算,结果是 9!

## 实现逻辑

技能会随机应用以下"怪异规则"之一:
- 加法(+) → 乘法(×)
- 减法(-) → 加法(+) 或 除法(÷)
- 乘法(×) → 加法(+) 或 减法(-)
- 除法(÷) → 减法(-) 或 乘法(×)

每次计算都会选择一个不同的"怪异"转换方式,让结果充满惊喜!
